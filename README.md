# ceezer-test-pz
Code challenge to apply to Ceezer by Patricio Zuniga

### How to run

1. Install pnpm

    `npm install -g pnpm`

2. Install dependencies

    `pnpm install`

3. Run migrations & data feed

    `pnpx prisma migrate dev`

    `pnpx prisma db seed`

4. Run apps

    `pnpm run dev`

5. Go to http://localhost:3000/
